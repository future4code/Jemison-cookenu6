## <h2>🔄 Rodando o projeto localmente</h2>  
  
Clone o projeto via HTTPS

```bash
  git clone https://github.com/future4code/Jemison-cookenu6.git
```

Entre no diretório do projeto

```bash
  cd Jemison-cookenu6
```

```bash
  npm install
```
